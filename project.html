Smart Parking System for UTAR, Kampar

  • Tools: Amazon Web Services (AWS)
  • Goals:
    • Enhanced the efficiency of vehicle authorization checks and improve the congestion.
    • Enhance the parking experience by helping students to locate available parking slots.
    • Gather and analyze data regarding traffic conditions to ease congestion and ensure that the parking resources are effectively used.
  • Technical implementation:
    • Utilized AWS IoT Core to establish connection between camera sensor and cloud services.
    • AWS S3 to store the photos from camera sensor.
    • Amazon Rekognition for license plates detection.
    • Constructed a database using DynamoDB to manage data regarding available parking slots.
    • Construted a web application hosted on Amazon EC2 to show the available parking slots at different zone.
    • Constructed a serverless function using AWS Lambda to ensure real-time update of data in web application.
    • Applied Amazon Athena to query and analyze data.
    • Create interactive dashboards and visualize data using Amazon QuickSight.
